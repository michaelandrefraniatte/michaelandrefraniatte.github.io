Any Channel Any Info

Watch French TV

The program is for watching Molotov.TV in a optimized way and full working manner, like fullscreen mode, or full resolution. It's enjoyable and can make you access faster all infos on French TV. It's the best of the two worlds between the Windows Store application and the website. It's possible to customize everything like removing invading ads. Let's see...

Considerations About the Program

I made this dedicated program for watching TNT on Molotov.TV named Playtov. You can set the link to open under a text file with same name of the program. It comes with another program, when having a laptop PC and Bluetooth speakers, to enable both. There's a parameters file under same folder of the program to enable it. To note, it ask to set a audio quality of 16 bits and 44100 Hz format for both audio cards, and enable both material acceleration and enhanced audio, under audio settings. It also need to set level of earing for laptop speakers to around 10 for avoid too loud sound. To note, if the program have issue, it need to delete the cache folder under same root. The program is located under bin/release folder from the code source folder in the dedicated repository. You can check here to find the right repository with same name.
