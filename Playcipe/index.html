Playcipe

Any Channel Any Video

Have Fun Watching Youtube Freely

The program is for watching Youtube freely without the need to be loggin. You have a home page to tweak for adding the channels you like or hate, even some other links, like search, or any other link. It's the best of the two worlds between the Windows Store application and the website. It removing invading ads by skipping it, cutting sound, displaying thumbnail instead. Let's see...

Considerations About the Program

I made this dedicated web browser named Playcipe. If the ads are too invasive, you can backward/forward the pages. I made a personnal recipe to add to favorites in Edge for surfing on Youtube with simple changes in HTML. I also made the Edge extension named recipe-extension. You can edit it with some editors I made, named CodeEditor and TextEditor. This file is explaining the way to get playlist id and channel id. This video can help you to create your Youtube data API key, in order to get channel id. The recipe page isn't to use as if, because it's very personal. It's the index.html file under assets folder where the program is. To note, there are handles to keys page up (to scroll up), page down (to scroll down), and F11 (to set Playcipe in fullscreen), and Escape (to close Playcipe). It comes with another program, when having a laptop PC and Bluetooth speakers, to enable both. To note, it ask to set a audio quality of 16 bits and 44100 Hz format for both audio cards, and enable both material acceleration and enhanced audio, under audio settings. There's a parameter in params.txt file under same folder of the program, to automatically launching. It also need to set level of earing for laptop speakers to around 10 for avoid too loud sound. To note, if the program have issue, it need to delete the cache folder under same root. I also made the Edge extension to install from edge://extensions in order to block invading ads on Youtube. But you will prefer AdBlock. The program is located under bin/release folder from the code source folder in the dedicated repository. You can check here to find the right repository with same name.
