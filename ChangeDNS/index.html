Considerations About the Program

The program is made for changing server DNS address automatically at launch against DNS attack. It run in real time, you just need to wait your network reconnect to internet after launching the program. It create automatically the right Windows firewall rule, but if it don't set the server DNS IP, you can set it manually under Windows parameters from network tab. The program is located under bin/release folder from the code source folder in the dedicated repository. You can check here to find the right repository with same name.
