Dedicated Standalone HID Competitive

This serie of programs is allowing to have inputs of physical HID and transfer to outputs of virtual HID for competitive gaming. It's easy to use and the program use scripting user interface. Let's see...

Considerations About the Program

You don't need any prerequist, just knowing to read and write some easy codes. You aiming or driving with competitive-immerssive-intuitive-natural-emotional controls with motion abilities, as well as pro player. I've been inspired, like with my custom program, I can set my favorite game to borderless for avoid crash, I can record it for upload on Youtube with a motion controller overlay in bottom center and a tracker pointer overlay in center, I stop useless process and services for optimization, I forward sound from my listeners to my HDMI TV speakers for having more gain on low frequencies and less gain on high frequencies, and I can play with my Wiimote for enjoy aiming and firing. The program is located under bin/release folder from the code source folder in the dedicated repository. You can check here to find the right repository with same name.
