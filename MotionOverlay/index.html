MotionOverlay

Motion Capture as Overlay in Games

Overlay in Your Games with Your Gaming Device Motions

The program enable an overlay for showing gameplay with gaming devices in action, while recording or streaming. Let's see...

Considerations About the Program

I've been inspired by this page I found on codepen.io. The Numpad keys are allowing to relocate position of overlay, change the size, the transparency, and even make disapearing the overlay. It only run with windowed mode of your game, not fullscreen (toggle with Alt + Enter). To note, there's file for parameters. For games in windowed mode but still displaying a border at top, use keys page down and page up to remove/add the border at top, when your game is the foreground window. The program is located under bin/release folder from the code source folder in the dedicated repository. You can check here to find the right repository with same name.
