Networking HID Data

Network HID Data Passthrough

This program is allowing to have inputs of physical HID and transfer to outputs of virtual HID from one PC to another. It's easy to use and the program can use scripting user interface. It's made for testing purpose, but it don't run very well for playing games. Let's see...

Considerations About the Program

You have to set your network card properties to private and allow outbound and inbound traffic for distant internal IP of both PC in Windows firewall. The program is located under bin/release folder from the code source folder in the dedicated repository. You can check here to find the right repository with same name.
