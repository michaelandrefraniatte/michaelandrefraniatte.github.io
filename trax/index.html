Be Like a Pro

Dedicated Standalone Screen Tracker

This program is allowing to have inputs of physical HID and transfer to outputs of virtual HID for screen tracking in flight games or racing games. It's easy to use and the program use configuration interface. A IR helmet is needed tracking by a Wiimote, and the keyboard/mouse is used as a Xbox controller. Let's see...

Considerations About the Program

You don't need any prerequist, just knowing to read and write some easy config files. Pressing both buttons 1 and 2 of Wiimote is connecting it automatically at program launch. It use setting files needing to set decimal instead of comma for number separator in regional advanced options under control panel. Pressing button A of Wiimote enable/disable outputs in order to change settings. It's possible to have different settings for different games by creating different name game folders, renaming the program for create different shortcuts in the desktop. The program is located under bin/release folder from the code source folder in the dedicated repository. You can check here to find the right repository with same name.
