Get Audio, Video, Embed Files of Youtube Channels

The program is able to retrieve the videos of a channel or a playlist with link of each video to download it in a raw or one by one. You have the choice between 3 types of files, the embed, the audio and the video. It also download the thumbnail of each video. Let's see...

Considerations About the Program

The program come with another program in order to have the same name of thumbnail and file associate. It also remove the thumbnails if the files associate were not been able to be downloaded. This video can help you to create your Youtube data API key, to write in the file apikey.txt under same folder of the program. You can test the process with Listing Videos. To note, channel ID start with UC but ID from playlist videos of a channel start with UU (after the ID is the same), and sometimes there's no videos access with ID from playlist videos of a channel but there's videos access with ID of playlists of a channel. This file is explaining the way to get playlist ID and channel ID. The program is located under bin/release folder from the code source folder in the dedicated repository. You can check here to find the right repository with same name.
