TextEditor

Edit Text File

Like Notepad without Security Holes

The program is like Notepad but not disappearing magically and without security holes. It remember your last file opened to open it at start and indicate the file opened. It have all features like Notepad, open, save, save as, search, replace, and edit/copy/cut/paste text. And also all shortcuts are running. Let's see...

Considerations About the Program

It only create a file automatically to remember the path of the last file opened. Otherwise, there isn't any issues and security holes. It never crash, and don't disappear magically. The program is located under bin/release folder from the code source folder in the dedicated repository. You can check here to find the right repository with same name.
