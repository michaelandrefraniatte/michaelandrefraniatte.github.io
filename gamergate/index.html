Gamer Gate for Addicted Gamers

Game Launcher for Every Games

The program is made for launching games from file shortcut links. You can customize all colors, and background image, as you want. Host somewhere sure a game image, add a game name, a well placed and well named game shortcut link, and the game image link, and it goes. You can't find something more easy to use, and more easy to launch all your games. Are you sure you can find something able to do just what it means to do? I think it's what everybody need, or maybe someone would do it, if it was not done ! I don't know, but here it is. Let's see...

Considerations About the Program

The folder of the program contains a background image gif file and a colors text file for customize it. It loops through each game image every 4 seconds. Click on a game name to view the game image, and click on the game image for launch the corresponding game. All game images you add, present on your computer, must be called with absolute path. Take example on path when you open a file in a web browser, the path is present in the address bar, when you click inside. But you will prefer Playnite. The program is located under bin/release folder from the code source folder in the dedicated repository. You can check here to find the right repository with same name.
