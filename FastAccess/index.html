Considerations About the Program

I made this page to edit by toggle mode with right click opening a menu. It's for having a page to fastly access websites with boxes. You can set the websites and name of the websites with easy edit. You can drag and drop a box when you toggle mode to edit. To note, if you encounter issue, simply open a empty file or set to default the background. You can use google drive to upload a background picture and use directly the link from copy link under google drive file menu when clicking right on it and choosing share. This page is here and the showcase here. I also made the Edge extension to install from edge://extensions in order to popup page in a new tab. But you can use the extension custom new tab, or my other own extension made for it, or my full extension for automatically have startup page and new tab page.
