WakeUpNow

Any Time Any Space

Have the Time Remaining and Be Advertise

The program is for wakeup in music and have time remaining before be alerted. You can change the music, the color of timers, and background image, as you want. If you have something to do important, or somehow forced by something or someone because you engaged you, it's the program to own. Don't be worry you can wakeup with a hunting music if you want, like a bugle, why not? Let's see...

Considerations About the Program

It's easy to use. I made the web version of wakeupnow. I also made the Edge extension named wakeup-extension. The extension can be installed from edge://extensions/. Take care to always open the extension under same site because it use localstorage for saving the time to wakeup. You can tweak a custom ring file with several music using the following cmd prompt command: copy /b *.mp3 wakeup.mp3. You can add silence between each music using the following silence files here. The program is located under bin/release folder from the code source folder in the dedicated repository. You can check here to find the right repository with same name.
