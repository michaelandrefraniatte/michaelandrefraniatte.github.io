SongCapture

Capture from Sound Card the Audio

Easy Tool to Capture Audio from Sound Card

The program is made for capture audio sound from sound card for recording perfectly song over internet. The file recorded is a file in MP3 format. Let's see...

Considerations About the Program

It's easy to record sound, the key numpad decimal (.) is starting the capture, and the key numpad 0 is stoping the capture, but you can use the buttons. After the end of capture, after you stop it, the folder, where the MP3 format file is created with your capture, is automaticaly open. The program is located under bin/release folder from the code source folder in the dedicated repository. You can check here to find the right repository with same name.
