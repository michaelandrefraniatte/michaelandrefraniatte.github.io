AimSight

A Crosshair for Top Down Shooters

Play with a Controller in Top Down Shooters

This program is allowing to have a custom crosshair for top down shooters playing with a controller because it often lack one when playing with a controller. You can change the crosshair before starting the program. It's just the file you tweak to replace under same folder of the program. Let's see...

Considerations About the Program

You shall use a GIF file, it will be a crosshair animated. The file is under assets folder where the program is. Use keyboard key shortcuts Alt + Tab to browse windows and Alt + F4 to close current window. To see it, your game, you are playing, must be in windowed mode. For games in windowed mode but still displaying a border at top, use keys page down and page up to remove/add the border at top, when your game is the foreground window. The program is located under bin/release folder from the code source folder in the dedicated repository. You can check here to find the right repository with same name.
