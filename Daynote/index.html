Daynote

Easy Notes for All Days

Take All Notes for All Years for All Days You Want to Take Notes

With this program, you have easy to remember everything to do all day and year long. It's just a reminder of your notes or tasks. Navigate between years and see all your year task or note number. If you have works to accomplish, it's the program, to handle. Take notes easy, edit it, add it, remove it, see all your notes of a day with simplicity. You have no chance to not work with it, or forget an anniverssary. Someone let you something to do, or you have taken his tasks, so don't forget to do it. If you forget to do it, the someone will never forgive you because this program exists. You can change the background color for better notes and days. It's here! Let's see...

Considerations About the Program

The folder of the program contains a colors text file for customize it. You can add without any limits. I made the web version of daynote. I also made the Edge extension named daynote-extension. The extension can be installed from edge://extensions/. Take care to always open the extension under same site because it use localstorage for saving tasks and notes. The program is located under bin/release folder from the code source folder in the dedicated repository. You can check here to find the right repository with same name.
