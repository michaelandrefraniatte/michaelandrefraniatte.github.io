Considerations About the Program

I made this dedicated web browser for listen music on Spotify named Playtify. For this program, under same folder, there's a text file for replace my profile page link by yours and a colors text file for customize audio visualization I added. To note, there's handle to key F11 to set Playtify in fullscreen. It comes with another program, when having a laptop PC and Bluetooth speakers, to enable both. There's a parameters file under same folder of the program to enable it. To note, it ask to set a audio quality of 16 bits and 44100 Hz format for both audio cards, and enable both material acceleration and enhanced audio, under audio settings. It also need to set level of earing for laptop speakers to around 10 for avoid too loud sound. If not using EchoBoost, tale care to have speakers with level of earing set to low, and your PC with level of earing set to high, otherwise the sound is too loud. To note, if the program have issue, it need to delete the cache folder under same root. The program is located under bin/release folder from the code source folder in the dedicated repository. You can check here to find the right repository with same name.
