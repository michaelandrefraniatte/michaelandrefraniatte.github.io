AudioVision

A Audio Visualizer as a Overlay

Play with Audio Visualization

This program is allowing to have a custom audio visualizer as an overlay for playing your game. It's for competitive gaming, there isn't any cheat like read and write in process memory, so it's not cheating. Let's see...

Considerations About the Program

You don't need to use Stereo Mixer. The audio in left and right speakers are separate for knowing from where comes ennemy. To see it, your game, you are playing, must be in windowed mode. For games in windowed mode but still displaying a border at top, use keys page down and page up to remove/add the border at top, when your game is the foreground window. The program is located under bin/release folder from the code source folder in the dedicated repository. You can check here to find the right repository with same name.
