Filter for Gaming

Filter Game Overlay

This program is allowing to have a filter over a game as an overlay. It can be pixelate or grayscale or whatever like blur. Let's see...

Considerations About the Program

You shall set the window title of the game you want to display with filter. You can choose to pixelate game or to have grayscale game to set in the file params.txt under same folder of the program. Use keyboard key shortcuts Alt + Tab to browse windows and Alt + F4 to close current window. To see it, your game, you are playing, must be in windowed mode. For games in windowed mode but still displaying a border at top, use keys page down and page up to remove/add the border at top, when your game is the foreground window. You must change game resolution under game options to remove the black border. The program is located under bin/release folder from the code source folder in the dedicated repository. You can check here to find the right repository with same name.
