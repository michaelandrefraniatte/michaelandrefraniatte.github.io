A Virtual Keyboard Made to Replace a Physical Keyboard

The program is a simple virtual Keyboard for replacing a physical Keyboard. It don't need to install anything, and have all main keys to type in any window without the need of a physical Keyboard. Let's see...

Considerations About the Program

If you want to open session, under Ease and Access, click on Keyboard to open the virtual Keyboard. Click on options of the virtual Keyboard to enable numpad by checking the box to allow it. Type your pin code with virtual numpad. The program is located under bin/release folder from the code source folder in the dedicated repository. You can check here to find the right repository with same name.
