Encryted Pages with Password Access

Your Pages with a Password for Access It

If you want your Pages to be entered with a password, I made it possible with a program. The program can encrypt pages and contents. You can have a nice page asking a password you define, for users to be able to see the page behind. Let's see...

Considerations About the Program

It's made for not having to charge a server with a database. A database isn't more secure than my method for needing to authentify users for just they can see some contents. It's cheaper and enough secure for some contents. In the example, coming with the program, the password is "salt", without the quotes. It can be used to secure the acces to an e-book. I also made the program named EbookViewver for sale e-book. The program is located under bin/release folder from the code source folder in the dedicated repository. You can check here to find the right repository with same name.
