AimAmplifier

Aim Ennemies Not Allies

Central Zone Zoomed as Overlay in FPS

This program is allowing to have a zoom of central zone as overlay in FPS games for competitive gamers. With it, you can play hardcore match without being kicked, because you killed an allies. You can recognize, easy peacy, ennemies between allies. It's for competitive gaming, there's no cheat like read and write in game process, so it's not cheating. Let's see...

Considerations About the Program

The Numpad keys are allowing to relocate position of overlay, change the size, the transparency, and even make disapearing the overlay. It only run with windowed mode of your game, not fullscreen (toggle with Alt + Enter). For games in windowed mode but still displaying a border at top, use keys page down and page up to remove/add the border at top, when your game is the foreground window. The program is located under bin/release folder from the code source folder in the dedicated repository. You can check here to find the right repository with same name.
