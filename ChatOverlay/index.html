ChatOverlay

Overlay of Chat

Youtube Chat While Streaming

The program enable an overlay for showing Youtube chat while streaming. It's made for streaming to see the chat of your stream while playing. Let's see...

Considerations About the Program

There's a file for parameters. You need to set a Youtube data API key and your channel ID, and to accept Youtube RGPD cookies, and then restart the program. The keyboard key Add is used to start the process for let time to retrieve the page with chat of your stream. Use the keyboard key Pause to not have sound of the page. It only run with windowed mode of your game, not fullscreen (toggle with Alt + Enter). For games in windowed mode but still displaying a border at top, use keys page down and page up to remove/add the border at top, when your game is the foreground window. You need to use the shortcut Alt + Tab to set the game in foreground for toggle the mode. The program is located under bin/release folder from the code source folder in the dedicated repository. You can check here to find the right repository with same name.
