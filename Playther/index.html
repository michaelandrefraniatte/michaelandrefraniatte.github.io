Watch Weather on Your Desktop

The program is for watching weather in a optimized way on desktop. It's the best of the two worlds between the Windows Store application and the website. Let's see...

Considerations About the Program

I made this dedicated program for watching weather on desktop named Playther. I was inspired by weatherwidget.io and weatherwidget.org. To note, the keyboard key Add is used to reload the page. The program is located under bin/release folder from the code source folder in the dedicated repository. You can check here to find the right repository with same name.
