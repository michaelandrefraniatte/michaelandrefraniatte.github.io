OSEF

Optimized and Simple Enhanced Framework

PHP Framework for Creating Websites Page to Page with Native Codes

I made a prototype framework for coding page to page websites, in order to demonstrate that native codes are more easy to use, than known frameworks not allowing communication between front-end and back-end in both ways. Let's see...

Considerations About the Program

It's more and more difficult to find native codes in web development, because it's more and more packaged with more and more frameworks. Everybody want to create their own framework, but they forgive security and optimization, but also developer and user learning. Often it's just a copy/paste, with bugs and obscelete codes, without taking count of big groups implemented since the start with amazing tools. I also made a cookie manager with a particular workout I invented to discharge third party scripts, as a Drupal module to adapt for osef.
